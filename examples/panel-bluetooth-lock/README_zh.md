## 家用门锁pro案例

- 产品名称 家用门锁pro

### 快速开始：

1. 前往[IoT 平台](https://iot.tuya.com/)创建产品

2. 选择品类 `家用门锁pro 选择蓝牙协议`
   ![功能选择](./images/iot1.png)
3. 功能选择

   必须的功能点

   ```
   remote_no_pd_setkey,
   remote_no_dp_key,
   ```

4. 安装项目并启动项目

   ```
   yarn && yarn start
   ```
